Geb, the Apocrypha Edition
================

## Getting Started

Install [GVM](http://gvmtool.net/)

```
> curl -s get.gvmtool.net | bash
```

Install Groovy

```
> gvm use groovy

Stop! groovy 2.1.7 is not installed.  

Do you want to install it now? (Y/n): Y 
```
